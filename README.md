<h2 style="color: #a855f7; font-family: monospace;">🚀 Key Features</h2>

<!-- Feature 1 -->
<div style="background-color: #0f0518; padding: 15px; border-radius: 8px; border-left: 4px solid #9333ea; font-family: monospace; color: #e9d5ff;">
  <p style="margin: 5px 0;"><strong>📊 Data Visualization: Seeing the Sound</strong></p>
  <ul style="margin-top: 5px; color: #c4b5fd; font-size: 0.95em;">
    <li><strong>Time-Domain Waveforms:</strong> Visualizing how signals evolve over time (e.g., the vibration of a bell or a tuning fork).</li>
    <li><strong>Frequency-Domain Spectrums:</strong> Using Fast Fourier Transforms (FFT) to decompose complex sounds into their individual frequency components (harmonics).</li>
    <li><strong>Signal Comparisons:</strong> Visualizing the difference between simple sinusoids and complex, multi-tonal instruments.</li>
  </ul>
</div>

<br>

<!-- Feature 2 -->
<div style="background-color: #0f0518; padding: 15px; border-radius: 8px; border-left: 4px solid #9333ea; font-family: monospace; color: #e9d5ff;">
  <p style="margin: 5px 0;"><strong>🎧 Audification: Hearing the Math</strong></p>
  <ul style="margin-top: 5px; color: #c4b5fd; font-size: 0.95em;">
    <li><strong>Signal Synthesis:</strong> Generating sounds using CosSignal and SinSignal with custom frequencies, amplitudes, and phases.</li>
    <li><strong>Timbre Exploration:</strong> Demonstrating how the "character" of a sound changes when different harmonics are added.</li>
    <li><strong>Wave Generation:</strong> Converting mathematical arrays into .wav formats for real-time playback.</li>
  </ul>
</div>

<br>

<h2 style="color: #a855f7; font-family: monospace;">📂 Project Structure</h2>

<!-- Module 1 -->
<div style="background-color: #0f0518; padding: 15px; border-radius: 8px; border-left: 4px solid #9333ea; font-family: monospace; color: #e9d5ff;">
  <p style="margin: 5px 0;"><strong>🔹 Module 1: Periodic Signals</strong></p>
  <ul style="margin-top: 5px; color: #c4b5fd; font-size: 0.95em;">
    <li><strong>Analysis:</strong> Exploring fundamental frequencies and periodicities (e.g., analyzing a 2.3ms period).</li>
    <li><strong>Synthesis:</strong> Combining multiple sine and cosine waves to create complex "rich" sounds.</li>
    <li><strong>Spectral Analysis:</strong> Identifying the dominant frequencies that give a sound its unique pitch.</li>
  </ul>
</div>

<!-- Module 2 -->
<div style="background-color: #0f0518; padding: 15px; border-radius: 8px; border-left: 4px solid #9333ea; font-family: monospace; color: #e9d5ff;">

  <ul style="margin-top: 5px; color: #c4b5fd; font-size: 0.95em;">
